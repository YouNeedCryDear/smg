## Tutorial Format

Each tutorial includes:

1. **Overview** — What you'll build and learn
2. **Prerequisites** — Required setup and knowledge
3. **Steps** — Detailed walkthrough with explanations
4. **Verification** — Confirm your setup works correctly
5. **Clean Up** — Remove tutorial resources
6. **Next Steps** — Where to go from here

## Before You Begin

Tutorials assume you have:

- [ ] Basic familiarity with command line operations
- [ ] Docker installed (for local tutorials)
- [ ] Kubernetes cluster access (for production tutorial)
- [ ] Text editor for configuration files

If you're new to SMG, start with the [Getting Started tutorial](getting-started.md).
